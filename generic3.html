False Colour Composite

The map above shows a false colour composite of a map. Looks different to your regular map doesn’t it, it’s almost like a piece of art! Although it looks difficult I can assure you that this map is actually very simple to make.

On Arc GIS Pro as I carried out a land cover classification of an area of Utrecht, Netherlands. The true colour composite is Red: Band 1, Green: Band 2, Blue: Band 3. To get the funky cool map above I simply switched the bands to; Red: Band 3, Green: Band 8, Blue: Band 5.

This made all the water in the map so the canals go black, green areas such as field go neon green, and developed areas went neon purple creating a piece of art that still sharply highlights the land usage.

Unsupervised Object Based Classification

The map shown here is of an area in Utrecht, the same area shown in the False Colour Composite Map above. This map has been generated using an unsupervised object based classification in Arc GIS Pro. Object based classification classifies an image based on groups of pixels that are alike.

For those of you who want to know how to render this kind of map, you first select your region of interest on google earth engine, I used Java script to code, but I think there is also oppurtunity for python. You then import this into Arc GIS and it will give you your area of interest on the map. On images Tab in Arc GIS pro you selct the cl;assification wizard, a window will appear. You select Unsupervised Classification Type and Object-based for classification, and leave the classification scheme as default scheme. Then you click run and will render the map shown.

A strength of using this classification method is that it is uncomplicated meaning that the steps to creation are simple. However uncomplicated it is, the map rendered is quite blurry and the colour produced are dark, therfore making it difficult to undwrstand and view different land usages.
