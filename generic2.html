Choropleth

The map above is a choropleth. Choropleth's use color shading to represent data values across geographic areas. This map demonstrates the data of United Nations Sustainable Development Goal 15.1.1, which stands for â€˜forest area as a total land area'. This basically means that it measures the proportion of the world's forested land area. This SDG helps identify unsustainable land practices, aiding forest protection and restoration.

This map was created using ARC GIS Online, the data for SDG 15.1.1 used in the creation of this map is from the UN's data bases. I imported this into ARC GIS, filtered any null data out, and started creating my choropleth. As you can see on the map above there is alot of missing data from countries but I still believe that the map turned out well. The map is interactive so tap on any region to be given details!
