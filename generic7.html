Original-3D Map

The map above has been created using LiDAR Data, LiDAR uses laser pulses to measure distances, mapping structures in three dimensions. To create this map I downloaded data from Geotiles, I chose a tile in amsterdam as I thought it would be interesting to see all of the infrastructure in 3D.

My data was originally in LAZ form so I converted it to LAS and added it to the map and then converted my map into -3D.After running this, the map rendered was the one above.
